SELECTOR $():

To select element do: $(elemName)

EX: $('div') OR $('p')


To select class do: $(className)

EX: $('.padded') OR $('.btn-success')


To select id do: $(idName)

EX: $('#map') OR $('#adminPanel')

Functions:

.after()

Appends HTML code after elemnt


For example here is HTML:

<p>Hello</p>

After running: $('p').after(' World')

You will get:

<p>Hello</p> World



.append()

Inserts HTML code at end of elements content


For example here is HTML body:

<p>Hello</p>

After running: $('p').after(' World')

You will get:

<p>Hello World</p>
