SELECTOR $():

To select element do: $(elemName)

EX: $('div') OR $('p')


To select class do: $(className)

EX: $('.padded') OR $('.btn-success')


To select id do: $(idName)

EX: $('#map') OR $('#adminPanel')

Functions:

.html()

Returns the HTML content of an element

OR if you call it with a string parameter you can assign to contents


Ex1

For example here is an HTML body:

<div><p>Hello World</p></div>


After Running: var a = $('div').html()

A would have the value: "<p>Hello World</p>"


Ex2

Another example HTML body:

<div><p></p></div>


After Running: var a = $('div').html('Hello World')

A would have the value: "<p>Hello World</p>"



.after()

Appends HTML code after elemnt


For example here is HTML:

<p>Hello</p>

After running: $('p').after(' World')

You will get:

<p>Hello</p> World



.append()

Inserts HTML code at end of elements content


For example here is HTML body:

<p>Hello</p>

After running: $('p').after(' World')

You will get:

<p>Hello World</p>

.remove()

Removes the element and returns it's HTML


For example here is an HTML body:

<p>Hello World</p>

Running:
console.log($('p').remove()); would output: "<p>Hello World</p>" to the console
and the HTML body would be empty.



For more functions, goto http://api.jquery.com/

Chaining

All jQuery functions return the selected jquery element. This allows you to chain like so:

$('p').before('Hello').html(' ').after('World');


Running that script on the following HTML:

<p></p>

Would result in:

Hello<p> </p>World
